# SETAP-B

[![Documentation Status](https://readthedocs.org/projects/setap-b/badge/?version=latest)](https://setap-b.readthedocs.io/en/latest/?badge=latest)

A comprehensive pet health and wellness management system featuring a FastAPI backend and Flutter mobile frontend.

## Documentation

📚 **[Read the Documentation](https://setap-b.readthedocs.io/)**

## Quick Start

### Backend Setup

```bash
cd petsync_backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirement.txt
python main.py
```

### Frontend Setup

```bash
cd frontend/maincode
flutter pub get
flutter run
```

## Project Structure

- **petsync_backend/** - FastAPI REST API
- **frontend/maincode/** - Flutter mobile application
- **docs/** - Sphinx documentation

## Features

- User authentication and management
- Pet profile management
- Health record tracking
- Feeding schedule management
- Health reports and analytics
- Veterinarian contact management
