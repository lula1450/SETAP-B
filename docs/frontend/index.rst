Overview
--------

The SETAP-B frontend is a cross-platform mobile application built with Flutter, supporting both iOS and Android.

Key Features
~~~~~~~~~~~~

- User authentication
- Pet profile management
- Dashboard with pet overview
- Health records tracking
- Feeding schedule management
- Vet contact management
- Health reports and analytics
